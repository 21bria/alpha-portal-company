# Alpha Portal Company

Modern corporate company portal built with Nuxt 4, TailwindCSS, and Django REST API.

## Features

- Corporate landing pages
- Company profile
- News & article portal
- Media gallery
- Project showcase
- Career & job vacancies
- Responsive modern UI
- SEO friendly pages
- Dark mode support

## Tech Stack

- Nuxt 4
- Vue 3
- TailwindCSS 4
- Shadcn Vue
- Pinia
- Django REST API

## Development

Install dependencies:

```bash
pnpm install
```

Run development server:

```bash
pnpm dev
```

Build for production:

```bash
pnpm build
```

## Environment Variables

Create `.env` file:

```env
NUXT_PUBLIC_API_BASE_URL=http://localhost:8000/api
```
